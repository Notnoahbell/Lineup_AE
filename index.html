Align
Alignment
Snaps selected layers to the left, center-H, right, top, center-V, or bottom of the target.
Align toSelection aligns to combined layer bounds; Composition to comp edges and center.
MarginExtra offset (px or %) added after aligning.
Offset KeysWhen on, keyframes move with the layer during alignment.
Distribute
Distribute
Six modes: horizontal spacing, vertical spacing, Z depth (3D), along path, radially, and grid layout.
Distribute toBoundary — Composition, Selection bounds, or a fixed Key Layer.
SpacingGap (%) between items for H/V modes.
RadiusCircle size for radial distribution.
RotateOrients each layer outward along its path tangent or radial angle.
Grid W × HColumn and row count for grid layout.
Align EdgesPins outermost layers to the boundary edges; distributes the rest evenly between them.
Anchor Point
3×3 grid Moves the anchor to any corner, edge midpoint, or center of the layer — without shifting the layer visually.

Create Rig
Generates null controllers linked by expressions to each vertex of the selected shape path.
Remove Rig
Strips the rig expressions and deletes the associated null layers.

Auto Crop
Crop buttons
All three buttons resize the selected precomp layer(s) to remove unused canvas space. Layer positions inside the inner comp shift to compensate, and anchor points and masks on the outer layer are preserved in comp-space.
Auto Crop
Shrinks the inner comp to the minimum bounding box of all visible layers at the current comp time.
Crop Max Area
Samples layer bounds at every keyframe time plus midpoints between keyframes to find the maximum area occupied across the full animation. Alerts and aborts if any transform property has an expression.
Mask Crop
Crops to the bounding box of the first enabled Add mask on the precomp layer, then removes that mask. Useful for framing a comp from a rough painted mask.
PadAdds extra pixels of breathing room around the cropped bounds. Without Expand, padding is clamped to stay within the original comp dimensions.
ExpandAllows the crop (and padding) to extend beyond the original comp bounds — use when content or handles reach outside the original canvas.
Layer Sort
Sort Reorders selected layers in the comp hierarchy by the chosen property and direction.
Property
Sort by Position, Rotation, or Scale.
AxisWhen sorting by Position, picks the axis to compare (X, Y, or Z).
Direction
Right (↑) = ascending low→high; left (↓) = descending high→low.
GroupParents all sorted layers under a new "Sort Group" null after sorting.
Organize
Buttons
Project-level tools for cleaning up, batch-editing, and exporting compositions.
Duplicate Comp (Deep)
Creates a fully independent copy of a comp, including every nested precomp inside it — so editing the copy never touches the original or any of its parts. Names are versioned (Comp → Comp_2 → Comp_3…) and each duplicate is placed in the same Project panel folder as its original. A comp used more than once inside the tree is only duplicated once and shared by every reference, mirroring the original structure. If a precomp layer is selected in the active comp's timeline, that layer is swapped to use the new duplicate in place; otherwise it duplicates whichever comp(s) are selected in the Project panel.
Decompose
Copies all layers from the selected precomp into the current comp. Creates a guide shape layer (named after the precomp) that carries the precomp's transform and parents all the inner layers to it. The original precomp layer is removed.
Consolidate Project
Scans the entire project for exact duplicate compositions (same name, size, frame rate, layers, and keyframes) and merges them into one, updating all usages. Also consolidates duplicate footage items and identical solid layers.
Batch Comp Settings
Applies dimensions, pixel aspect, frame rate, resolution, duration, and/or start time to every selected composition at once. Width/Height have a scrubbable field and an aspect-ratio lock; the comp list at the top can be trimmed (✕) before applying.
Batch Rename
Renames every selected composition from a pattern. [#], [##], [###]… insert a zero-padded index counting up from Start at, in the order shown — drag a row's handle to reorder and renumber, or remove (✕) a comp to skip it. There's also a semi-secret [A] token: it inserts a letter (A, B, … Z, AA, AB, …) based on position alone, always starting at A no matter what Start at is set to.
Comp Export
Combines After Effects' own Save, Save As, and Reduce Project: saves the current project as-is, duplicates it to a new file, and strips the duplicate down to just the selected compositions, leaving asset links pointing at their current locations (nothing is copied or collected). The new reduced project is left open. Suggested filename is ProjectName_CompName when one comp is selected, or ProjectName_reduced for more than one; Browse to pick a different name or folder. Requires the project to already be saved once.
