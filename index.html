Help
Align
Alignment
Snaps selected layers to the left, center-H, right, top, center-V, or bottom of the target.
Align toSelection aligns to combined layer bounds; Composition to comp edges and center.
Properties buttonThe sliders icon at the end of the button row opens Margin (extra offset, px or %) and Offset Keys (keyframes move with the layer during alignment). The checkmark in its top-right corner just closes it — edits save immediately.
Keyframe modeSelect keyframes in the Timeline and Left/Center/Right shift them as a group instead of moving layer position (hourglass badge marks the edge buttons; Center swaps to a playhead glyph). Multiple keyframes on the same property move together as one rigid block — aligned to the earliest/playhead/latest keyframe in the selection (Selection), or the playhead/comp center/comp end (Composition). Top/Center-V/Bottom don't apply to keyframes.
Distribute
Distribute
Six modes: horizontal spacing, vertical spacing, Z depth (3D), along path, radially, and grid layout.
Distribute toBoundary — Composition, Selection bounds, or a fixed Key Layer.
Key Layer modeHorizontal, Vertical, Z, and Radial always line layers up back to back against the key layer when in Key Layer mode. Path has its own draggable spacing in its right-click menu.
Right-click Z / Path / Radial / GridOpens a small menu with that mode's own settings, so the main panel only shows the controls that always apply.
Properties buttonThe sliders icon at the end of the button row lists every Z, Path, Radial, and Grid setting as plain fields in one place — no draggable previews. Edits save immediately; the checkmark in its top-right corner (or Escape, or clicking elsewhere) closes it.
Z menuDrag the depth track to set the Z range, or check Step for fixed spacing instead of even distribution.
Path menuDrag along the curve (or type a value) to set spacing % in Key Layer mode — the squares preview where items land. Rotate orients each layer along the path tangent, independent of Radial's Rotate.
Radial menuDrag in the preview or type a value to set the circle's radius. Rotate orients each layer outward along the radial angle.
Grid menuClick a cell to set columns × rows, optionally override the horizontal/vertical gap, and toggle Align Edges to pin corner layers to the boundary while distributing the rest evenly between them.
Sizing
Horizontal / Vertical / Both
Scales each selected layer (or the Key Layer's followers) so its bounds match the target's width, height, or both. Scaling happens around each layer's own anchor point, so position is otherwise untouched.
Size toComposition matches the comp's dimensions; Selection matches the combined bounding box of all selected layers; Key Layer matches the topmost selected layer's bounds and resizes the rest of the selection.
StretchScales only the matched axis (or, on Both, fits width and height exactly — ignoring aspect ratio).
CropScales proportionally — Horizontal/Vertical also grow the other axis to match; Both scales uniformly by whichever axis needs the larger factor, so the layer fully covers the target (aspect ratio preserved, like CSS background-size: cover).
MoveCenters each layer's bounds in the target before scaling — the same behavior as After Effects' own Fit to Comp.
Anchor Point
3×3 grid Moves the anchor to any corner, edge midpoint, or center of the layer — without shifting the layer visually.
Based onObject positions within each layer's own source bounds; Selection within the combined bounding box of all selected layers; Composition relative to the full comp.
Copy / PasteSaves an exact anchor coordinate from one layer and applies it to others.
Clear
Empties the anchor clipboard.
NullCreates a null layer at the layer's current anchor position.
Link Property to Controller
Select the same property (Rotation, Opacity, Position, or an effect's Color/Angle/Point/Slider) on 2+ layers, then click to create a null with a matching controller effect (Angle/Point/Color/Slider Control) and link every layer to it via expression. Without Offset, all layers snap to the first layer's value; with Offset, each keeps its own value and the controller shifts them together.
Ease Copy
Copy Captures the Bezier easing handles from the selected keyframes.
EasePastes the saved easing curve onto selected keyframes.
ValuePastes only the saved keyframe value — the existing curve is unchanged.
Clear
Empties the ease clipboard.
Auto Crop
Crop buttons
Resizes the selected precomp layer(s) to remove unused canvas space, shifting inner-comp positions to compensate while preserving anchor points and masks on the outer layer.
Auto Crop
Shrinks the inner comp to the minimum bounding box of all visible layers at the current comp time.
Crop Max Area
Samples layer bounds at every keyframe time plus midpoints between keyframes to find the maximum area occupied across the full animation. Alerts and aborts if any transform property has an expression.
Mask Crop
Crops to the bounding box of the first enabled Add mask on the precomp layer, then removes that mask. Useful for framing a comp from a rough painted mask.
PadAdds extra pixels of breathing room around the cropped bounds. Without Expand, padding is clamped to stay within the original comp dimensions.
ExpandAllows the crop (and padding) to extend beyond the original comp bounds — use when content or handles reach outside the original canvas.
Layer Sort
Sort Reorders selected layers in the comp hierarchy by the chosen property and direction.
Property
Sort by Position, Rotation, or Scale.
AxisWhen sorting by Position, picks the axis to compare (X, Y, or Z).
Direction
Right (↑) = ascending low→high; left (↓) = descending high→low.
GroupParents all sorted layers under a new "Sort Group" null after sorting.
Organize
Buttons
Project-level cleanup, batch-editing, and export tools.
True Duplicate Comp
Fully independent copy of a comp and every nested precomp inside it, so edits never touch the original. Names are versioned (Comp → Comp_2…); a comp used more than once is only duplicated once and shared. Swaps a selected precomp layer to the new duplicate, or duplicates whichever comp(s) are selected in the Project panel.
Decompose
Copies a precomp's layers into the current comp under a new guide shape layer (named after it) carrying its transform, and removes the original precomp layer.
Consolidate Project
Merges exact duplicate comps (same name, size, frame rate, layers, keyframes) project-wide, updating every usage — plus duplicate footage and identical solids.
Configure Project Structure
Editable folder-structure builder — tabs hold named presets (Heist is the built-in default); + adds a preset, drag a row to reorder or nest it, each row's +/trash adds or removes folders. Apply sorts loose comps/assets into the active preset by file type, moving any pre-existing folder into its "imported" folder — check Break Structure to unpack those instead. Restore Default brings back stock Heist; Cancel discards the session's edits.
Batch Comp Settings
Two tabs. Settings applies dimensions, pixel aspect, frame rate, resolution, duration, and/or start time to every selected comp at once (scrubbable Width/Height with an aspect lock; Snap Keyframes rounds everything to the new frame rate). Rename applies a numbered pattern — [#]/[##]/[###] insert a zero-padded counter from Start at, drag rows to reorder/renumber, ✕ to skip one; the semi-secret [A] token inserts a letter (A…Z, AA…) by position alone, always starting at A.
Comp Export
Combines Save, Save As, and Reduce Project: saves as-is, duplicates to a new file, and strips the duplicate to just the selected comps (asset links untouched, nothing copied). Suggests ProjectName_CompName (or _reduced for multiple); requires the project already saved once.
Shape Tools
Select Paths / Fill / Stroke
Left-click reselects whichever mode you last picked (icon reflects it); right-click opens a flyout to switch between selecting all Paths, Fill colors, or Stroke colors on the selected shape layers.
Change Stroke Type
Cycles cap and join together; hold Alt to cycle just the cap.
Color Management
Consolidates every solid color on the selected layers (or the whole comp) into one Color Controller null, one Color Control effect per unique color — reuses an existing controller instead of making a new one.
Merge Shapes
Merges every selected shape layer into the first one selected, preserving each layer's current position, rotation, and scale.
Explode Shapes
Splits each selected shape layer's top-level Contents items into their own new layers, preserving comp-space appearance.
Fill / Stroke
Live swatches above the toolbar show the selection's fill and stroke color (and stroke width) — Mix when they differ. Click a swatch to set every instance in scope at once; click Fill/Stroke (or the sliders icon) to edit each shape individually, including turning its fill or stroke on or off. Drag the blue width number to scrub it, or click to type a value.
