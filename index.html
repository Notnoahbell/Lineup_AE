Help
Align
Alignment
Snaps selected layers to the left, center-H, right, top, center-V, or bottom of the target.
Align toSelection aligns to combined layer bounds; Composition to comp edges and center.
Properties buttonThe sliders icon at the end of the button row opens Margin (extra offset, px or %) and Offset Keys (keyframes move with the layer during alignment). The checkmark in its top-right corner just closes it — edits save immediately.
Distribute
Distribute
Six modes: horizontal spacing, vertical spacing, Z depth (3D), along path, radially, and grid layout.
Distribute toBoundary — Composition, Selection bounds, or a fixed Key Layer.
Key Layer modeHorizontal, Vertical, Z, and Radial always line layers up back to back against the key layer when in Key Layer mode. Path has its own draggable spacing in its right-click menu.
Right-click Z / Path / Radial / GridOpens a small menu with that mode's own settings, so the main panel only shows the controls that always apply.
Properties buttonThe sliders icon at the end of the button row lists every Z, Path, Radial, and Grid setting as plain fields in one place — no draggable previews. Edits save immediately; the checkmark in its top-right corner (or Escape, or clicking elsewhere) closes it.
Z menuDrag the depth track to set the Z range, or check Step for fixed spacing instead of even distribution.
Path menuDrag along the curve (or type a value) to set spacing % in Key Layer mode — the squares preview where items land. Rotate orients each layer along the path tangent, independent of Radial's Rotate.
Radial menuDrag in the preview or type a value to set the circle's radius. Rotate orients each layer outward along the radial angle.
Grid menuClick a cell to set columns × rows, optionally override the horizontal/vertical gap, and toggle Align Edges to pin corner layers to the boundary while distributing the rest evenly between them.
Sizing
Horizontal / Vertical / Both
Scales each selected layer (or the Key Layer's followers) so its bounds match the target's width, height, or both. Scaling happens around each layer's own anchor point, so position is otherwise untouched.
Size toComposition matches the comp's dimensions; Selection matches the combined bounding box of all selected layers; Key Layer matches the topmost selected layer's bounds and resizes the rest of the selection.
StretchScales only the matched axis (or, on Both, fits width and height exactly — ignoring aspect ratio).
CropScales proportionally — Horizontal/Vertical also grow the other axis to match; Both scales uniformly by whichever axis needs the larger factor, so the layer fully covers the target (aspect ratio preserved, like CSS background-size: cover).
MoveCenters each layer's bounds in the target before scaling — the same behavior as After Effects' own Fit to Comp.
Anchor Point
3×3 grid Moves the anchor to any corner, edge midpoint, or center of the layer — without shifting the layer visually.
Based onObject positions within each layer's own source bounds; Selection within the combined bounding box of all selected layers; Composition relative to the full comp.
Copy / PasteSaves an exact anchor coordinate from one layer and applies it to others.
Clear
Empties the anchor clipboard.
NullCreates a null layer at the layer's current anchor position.
Link Property to Controller
Select the same property (e.g. Rotation, Opacity, Position, or an effect's Color/Angle/Point/Slider) on 2 or more layers, then click to create a new null with a matching controller effect — Angle Control for rotation and other degree-based properties, Point Control for 2D points, Color Control for colors, and Slider Control for everything else. Every selected layer's property is linked to the controller with an expression. Without Offset, the controller is seeded with the first selected layer's value and every layer snaps to match it. With Offset, each layer keeps its own original value, and moving the controller shifts them all by the same amount.
Ease Copy
Copy Captures the Bezier easing handles from the selected keyframes.
EasePastes the saved easing curve onto selected keyframes.
ValuePastes only the saved keyframe value — the existing curve is unchanged.
Clear
Empties the ease clipboard.
Shape Rigs Beta
Create Rig
Generates null controllers linked by expressions to each vertex of the selected shape path.
Remove Rig
Strips the rig expressions and deletes the associated null layers.
Recalculate
Re-maps null positions along the path — use after editing the shape.
Auto Crop
Crop buttons
All three buttons resize the selected precomp layer(s) to remove unused canvas space. Layer positions inside the inner comp shift to compensate, and anchor points and masks on the outer layer are preserved in comp-space.
Auto Crop
Shrinks the inner comp to the minimum bounding box of all visible layers at the current comp time.
Crop Max Area
Samples layer bounds at every keyframe time plus midpoints between keyframes to find the maximum area occupied across the full animation. Alerts and aborts if any transform property has an expression.
Mask Crop
Crops to the bounding box of the first enabled Add mask on the precomp layer, then removes that mask. Useful for framing a comp from a rough painted mask.
PadAdds extra pixels of breathing room around the cropped bounds. Without Expand, padding is clamped to stay within the original comp dimensions.
ExpandAllows the crop (and padding) to extend beyond the original comp bounds — use when content or handles reach outside the original canvas.
Layer Sort
Sort Reorders selected layers in the comp hierarchy by the chosen property and direction.
Property
Sort by Position, Rotation, or Scale.
AxisWhen sorting by Position, picks the axis to compare (X, Y, or Z).
Direction
Right (↑) = ascending low→high; left (↓) = descending high→low.
GroupParents all sorted layers under a new "Sort Group" null after sorting.
Organize
Buttons
Project-level tools for cleaning up, batch-editing, and exporting compositions.
True Duplicate Comp
Creates a fully independent copy of a comp, including every nested precomp inside it — so editing the copy never touches the original or any of its parts. Names are versioned (Comp → Comp_2 → Comp_3…) and each duplicate is placed in the same Project panel folder as its original. A comp used more than once inside the tree is only duplicated once and shared by every reference, mirroring the original structure. If a precomp layer is selected in the active comp's timeline, that layer is swapped to use the new duplicate in place; otherwise it duplicates whichever comp(s) are selected in the Project panel.
Decompose
Copies all layers from the selected precomp into the current comp. Creates a guide shape layer (named after the precomp) that carries the precomp's transform and parents all the inner layers to it. The original precomp layer is removed.
Consolidate Project
Scans the entire project for exact duplicate compositions (same name, size, frame rate, layers, and keyframes) and merges them into one, updating all usages. Also consolidates duplicate footage items and identical solid layers.
Configure Project Structure
Opens an editable folder-structure builder. Tabs across the top hold named presets — Heist is the built-in default (1.COMPS/1.MASTER, 2.BUILD, 3.PRECOMPS · 2.ASSETS/3D_RENDERS, AE_RENDERS, ARTWORK/AI, ARTWORK/PSD, AUDIO, FOOTAGE, IMAGES · 3.IMPORTED/AE_PROJECTS, C4D_SCENES · 4.REFERENCES/BOARDS, OFFLINES, STYLEFRAMES); click + to add your own, double-click a tab to rename the preset. In the tree, click a folder's name to rename it, click the chevron to collapse/expand its contents, drag a row by its grip handle to reorder it (drop on the top/bottom edge of another row) or nest it inside another folder (drop in the middle), and use each row's + / trash to add or remove folders. Restore Default brings back the stock Heist structure to edit again; if every preset is deleted, it's recreated automatically. Apply builds whichever preset is active and sorts loose comps/assets into it by file type (PSD/AI/image/video/audio/C4D go by folder name — rename Heist's own folders freely and the matching still finds them, or name your own folders the same way for the same effect). Any pre-existing folder not part of the structure is moved as a whole into the structure's "imported" folder (Heist: 3.IMPORTED/AE_PROJECTS); check Break Structure to instead unpack those folders — every comp/asset inside is sorted individually and the emptied folders are deleted. Cancel discards all edits made in this session.
Batch Comp Settings
Applies dimensions, pixel aspect, frame rate, resolution, duration, and/or start time to every selected composition at once. Width/Height have a scrubbable field and an aspect-ratio lock; the comp list at the top can be trimmed (✕) before applying. When changing frame rate, Snap Keyframes rounds every keyframe and every layer's in/out points to the nearest whole frame at the new rate.
Batch Rename
Renames every selected composition from a pattern. [#], [##], [###]… insert a zero-padded index counting up from Start at, in the order shown — drag a row's handle to reorder and renumber, or remove (✕) a comp to skip it. There's also a semi-secret [A] token: it inserts a letter (A, B, … Z, AA, AB, …) based on position alone, always starting at A no matter what Start at is set to.
Comp Export
Combines After Effects' own Save, Save As, and Reduce Project: saves the current project as-is, duplicates it to a new file, and strips the duplicate down to just the selected compositions, leaving asset links pointing at their current locations (nothing is copied or collected). The new reduced project is left open. Suggested filename is ProjectName_CompName when one comp is selected, or ProjectName_reduced for more than one; Browse to pick a different name or folder. Requires the project to already be saved once.
